## Organització del projecte (Kanban)
- **Tauler Kanban** creat amb **GitHub Projects** amb columnes: *To do / In progress / Done*.
- Cada tasca és una **issue** amb labels i deadlines.
- Exemples de issues:
  - Crear estructura inicial (#1)
  - Formulari HTML (#2)
  - Validació JS (#3)
